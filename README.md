# Игра "Пятнашки"

Для предмета АМУ.

## В приложении используются технологии 👨🏻‍💻:
- Apple MVC 📚
- Frame animations 🔧

## Основные функции 🎯:
- Релизовать игру
- При генерации новой игры сделать проверку на решаемость (только 50% рандомно сгенерированных игр решаемы)
